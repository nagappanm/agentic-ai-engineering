# phaser-demo — app knowledge

Phaser 4 (2D/WebGL game engine); game objects addressed by name via the main camera sample for the klew **scene tier** (see `references/selector-policy.md`). The
objects have no DOM element, so they are cached as `tier: scene` and clicked via
the app's own scene model — never a hardcoded pixel. Sample: `e2e/scene/phaser.html`.

## Notes & gotchas

- Requires the scene instance (`window.__phaser` (the active Scene)) to be exposed.
- Every scene entry is `a11y_flag: true` — canvas objects are absent from the
  accessibility tree.
- Act via `scripts/scene_click.py` or the generated `phaser-demo.pom.ts`.
