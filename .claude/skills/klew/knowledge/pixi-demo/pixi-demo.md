# pixi-demo — app knowledge

PixiJS 8 (WebGL), display objects addressed by their label sample for the klew **scene tier** (see `references/selector-policy.md`). The
shapes have no DOM element, so they are cached as `tier: scene` and clicked via
the app's own scene model — never a hardcoded pixel. Sample: `e2e/scene/pixi.html`.

## Notes & gotchas

- Requires the scene instance (`window.__PIXI_APP__`) to be exposed.
- Every scene entry is `a11y_flag: true` — canvas shapes are absent from the
  accessibility tree.
- Act via `scripts/scene_click.py` or the generated `pixi-demo.pom.ts`.
