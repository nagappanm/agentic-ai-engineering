# chartjs-demo — app knowledge

Chart.js 4 (2D canvas), bars addressed by category label sample for the klew **scene tier** (see `references/selector-policy.md`). The
shapes have no DOM element, so they are cached as `tier: scene` and clicked via
the app's own scene model — never a hardcoded pixel. Sample: `e2e/scene/chartjs.html`.

## Notes & gotchas

- Requires the scene instance (`window.__chart`) to be exposed.
- Every scene entry is `a11y_flag: true` — canvas shapes are absent from the
  accessibility tree.
- Act via `scripts/scene_click.py` or the generated `chartjs-demo.pom.ts`.
