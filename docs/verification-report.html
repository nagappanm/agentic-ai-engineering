Do it yourself

Reproduce this report in three commands

Everything above is deterministic and offline. From a fresh clone of main:

clone → decision engine → live journeys
# 1 · the decision engine — no browser, ~10s
python -m pytest tests/test_pr_gate.py tests/test_author.py tests/test_author_nl.py -q
#   → 25 passed

# 2 · test-trust grading & traceability
cd testguard && npm install && npm test
#   → 33 passed (7 files)

# 3 · real journeys against the demo app
(cd e2e/app && python3 -m http.server 8123 &)
cd e2e && npm install && npx playwright install chromium
BASE_URL=http://127.0.0.1:8123 npx playwright test
#   → 13 journeys passed
